# Facial Interface Client

This project is a C++ client for a facial interface system, designed to provide a visual representative for Large Language Models. It captures microphone input, communicates with a processing backend to receive emotion-tagged audio, and renders a 3D character with synchronized facial expressions and speech playback using Raylib.

## Build Instructions

```bash
mkdir build
cd build
cmake ..
make
```


## Running Instructions

```
cd build
./FacialInterface
```

## Usage Instructions
Press the R button and then speak into the microphone, when finished speaking press R again wait for the response.
